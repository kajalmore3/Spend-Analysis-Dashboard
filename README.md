# 📊 Spend Analytics Dashboard

## 📌 Project Overview

The Spend Analytics Dashboard is an interactive Power BI dashboard designed to analyze organizational spending patterns across suppliers, plants, product categories, and monthly purchasing trends. The dashboard helps decision-makers monitor procurement performance, identify high-value suppliers, and optimize spending through interactive visualizations.

---

## 🎯 Objective

- Analyze overall organizational spending.
- Track monthly order quantity and net order value.
- Identify top suppliers based on order value.
- Monitor plant-wise procurement performance.
- Analyze product category-wise spending.
- Support data-driven procurement decisions.

---

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dataset contains procurement and spend-related information, including:

- Supplier Name
- Plant Name
- Category (L4)
- Order Quantity
- Net Order Value
- Order Date
- Product Category

---

## 📊 Dashboard Features

- 📌 Total Number of Plants KPI
- 🏢 Total Suppliers KPI
- 📦 Order Quantity by Category (Treemap)
- 💰 Net Order Value by Category (Donut Chart)
- 📈 Monthly Order Quantity Trend
- 📉 Monthly Net Order Value Trend
- 🏭 Plant-wise Total Order Value
- 🤝 Supplier-wise Total Order Value
- 🎛️ Interactive Slicers (Supplier, Category-L4, Plant)

---

## 📈 Key Insights

- Paper category contributes the highest share of total net order value.
- Plant P1 has the highest procurement value among all plants.
- Supplier S5 is the top-performing supplier based on total order value.
- Monthly spending trends help identify peak procurement periods.
- Interactive filters enable dynamic analysis by supplier, plant, and category.
